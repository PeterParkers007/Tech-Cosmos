## 🧭 指导原则

1. **操作系统优先** — 复杂领域先构建 OS 再构建玩法
2. **零耦合** — 卫星包互不引用，Adapter 集成
3. **Hub 优先** — 安装与脚手架统一入口
4. **用例驱动** — 技术演进须有演示项目验证
5. **文档即设计** — 本仓库与 Hub README 同步维护
